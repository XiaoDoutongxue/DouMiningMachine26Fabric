# DouMiningMachine 26.2 Fabric

本版本包含：挖矿机、挖石机、铲土机。

## 本次 fix15 更新

1. 运行速度燃料消耗已大幅提高。
   - 旧逻辑：燃料消耗倍率 = 区块数量 × 速度。
   - 新逻辑：燃料消耗倍率 = 区块数量 × 速度 × 速度。
   - 速度 10 时，单速度倍率就是 100 倍；如果区块也调高，煤炭会消耗得非常快。

2. 铲土机新增模式切换。
   - 同层模式：只铲和铲土机同一个 Y 层的泥土、草方块等。
   - 整区块模式：扫描设定区块内所有 Y 层的泥土、草方块等。
   - 在铲土机 UI 里点击“同层 / 整区”按钮切换。

3. 原有功能保留。
   - 三种机器都支持煤炭燃料。
   - 三种机器都支持 1-100 个区块范围。
   - 三种机器都支持 1x-10x 速度调节。
   - 三种机器都支持旁边箱子输出。
   - 挖矿机过滤槽支持矿石方块，也支持粗铁、粗金、粗铜、煤炭、钻石、绿宝石、红石粉、青金石。
